## Formato da Etiqueta

- Tamanho: 10cm x 5cm (283 x 142 pontos a 203 DPI)
- Código de barras: Code-128
- Fonte: Zebra 0 (padrão)
- Comandos ZPL utilizados:
  * ^XA / ^XZ - Início/fim do documento
  * ^GFA - Imagem gráfica (logo)
  * ^FO - Posicionamento de campo
  * ^A0N - Fonte padrão
  * ^BCN - Código de barras Code-128
